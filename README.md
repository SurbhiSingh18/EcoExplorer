# Eco Explorer

## Project Description

Eco Explorer is an Android-based eco-travel planning application that helps users discover sustainable travel destinations based on budget and preferred activities.

## Tech Stack:-

### Frontend
- Android XML
- Material UI Components

### Backend & Database
- Firebase Authentication
- Firebase Firestore

### Development Tools
- Android Studio
- Java
- Firebase Console

## Features:-

- User Registration & Login
- Guest Mode Access
- Eco-Friendly Trip Planning
- Budget-Based Destination Suggestions
- Activity-Based Recommendations
- Destination Summary Generation
- Sustainable Travel Tips
- Previous Trip History Management
